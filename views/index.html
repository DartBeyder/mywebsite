Yeezy 350 Zebra

Kanye West and adidas release their second 350 of 2017 with the release of the adidas Yeezy Boost 350 V2 "Zebra." The sneaker features a white and core black Primeknit upper with a red SPLY 350 accent and a translucent white midsole that houses full-length Boost. This is a very limited release that will be available at select adidas flagship stores, adidas.com, and Yeezy Supply.
